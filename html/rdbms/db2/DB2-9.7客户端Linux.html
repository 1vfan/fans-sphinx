1. DB2-9.7客户端¶

cd /opt/ibm/db2/V9.7/instance

[root@arcgis bin]# ./db2 SQL10007N Message “-1390” could not be retrieved. Reason code: “3”.

[root@arcgis instance]# ./db2idrop -f arcgis DBI1324W Support of the -f command is deprecated. For

more information, see the DB2 Information Center.

DBI1070I Program db2idrop completed successfully.

[root@arcgis instance]# ./db2icrt -u arcgis arcgis DBI1070I Program db2icrt completed successfully.

查看node

db2 list node directory

删除node信息

db2 uncatalog node xxx

1.1. db2客户端连接远程server¶

C:IBM32SQLLIBBIN> db2 catalog tcpip node nodegeo remote 192.168.30.106 server 60000 C:IBM32SQLLIBBIN> db2 catalog db geodb at node nodegeo C:IBM32SQLLIBBIN> db2 connect to geodb user sde using sde

db2ilist 查看所有实例 db2 get instance 查看当前连接的实例 db2 list db directory 查看当前实例下连接的数据库
