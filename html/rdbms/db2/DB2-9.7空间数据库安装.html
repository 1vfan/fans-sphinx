2. DB2空间数据库安装¶

2.1. 系统环境配置¶

  • hostname

  • iptables

  • selinux

$ vi /etc/sysconfig/network
HOSTNAME=db2

$ vi /etc/hosts
192.168.30.106 bgis.zjjzfy.com db2

# 永久关闭防火墙/开放特定端口供外部访问
$ service iptables stop
$ service iptables status
$ chkconfig iptables off

$ getenforce
$ vi /etc/sysconfig/selinux
SELINUX=enforcing -> SELINUX=disabled

$ init 6

2.2. 安装DB2 Server¶

  • 检查依赖

  • 安装

  • 导入许可

$ tar -zxvf v9.7fp11_linuxx64_server.tar.gz

$ ./db2prereqcheck
WARNING: The 32 bit library file libstdc++.so.6 is not found on the system.

$ rpm -ivh glibc-2.12-1.107.el6.i686.rpm nss-softokn-freebl-3.12.9-11.el6.i686.rpm
$ rpm -ivh libgcc-4.4.7-3.el6.i686.rpm
$ rpm -ivh libstdc++-4.4.7-3.el6.i686.rpm

$ ./db2_install
安装路径 no
数据库版本 ESE

$ cd /opt/ibm/db2/V9.7/adm
$ ./db2licm -a /opt/install/db2ese_c.lic
$ ./db2licm -l

2.3. 安装空间模块¶

  • 检查gse

  • 安装gse

  • 检查确认

$ db2ls -q -b /opt/ibm/db2/V9.7/
缺少**SPATIAL_EXTENDER_SERVER_SUPPORT**模块

$ tar -zxvf v9.7fp11_linuxx64_gse.tar.gz
$ ./gse/db2_install
安装路径 no

$ db2ls -q -b /opt/ibm/db2/V9.7/

2.4. 创建数据库实例¶

  • 创建实例用户组/用户

  • 创建实例

  • 创建表空间目录

  • 创建数据库

  • 创建缓冲池

  • 创建表空间

  • 创建临时表空间

  • 配置自启动

$ groupadd -g 1000 db2adm
$ useradd -g db2adm bgis
$ passed bgis

$ /opt/ibm/db2/V9.7/instance/db2icrt -u bgis bgis

$ mkdir -p /home/bgis/bgis_data/tbspaces/data_sms
$ chown -R bgis:db2adm /home/bgis/bgis_data

$ su - bgis
$ db2start
$ db2 create database geodb using codeset gbk territory cn
$ db2 connect to geodb

# 创建该数据库的页面大小为32K的缓冲池
$ db2 create BUFFERPOOL BP32 size 100000 pagesize 32k
# 创建页面大小为32K的表空间
$ db2 "create REGULAR TABLESPACE data_sms pagesize 32k managed by system using('/home/bgis/bgis_data/tbspaces/data_sms') bufferpool bp32"
# 创建页面大小为32k的临时表空间
$ db2 CREATE SYSTEM TEMPORARY TABLESPACE TEMPSPACE_bgis PAGESIZE 32K MANAGED BY AUTOMATIC STORAGE EXTENTSIZE 32 PREFETCHSIZE AUTOMATIC BUFFERPOOL bp32

$ ./instance/db2iauto -on bgis

2.5. 外部连接配置¶

  • 设置连接端口

  • 设置连接协议

  • 重启db2数据库

$ su - bgis
$ db2 get dbm cfg | grep SVCENAME
$ db2 update dbm cfg using SVCENAME 60000

$ db2set -all
$ db2set DB2COMM=TCPIP

$ db2 force application all
$ db2stop
$ db2start

2.6. 日志和堆参数设置¶

  • 查看数据库参数

  • 修改日志参数

  • 修改空间数据库堆参数

$ su - bgis
$ db2 connect to geodb
$ db2 get db cfg

$ db2 get db cfg | grep LOG
$ db2 update db cfg using LOGFILSIZ 100000
$ db2 update db cfg using LOGPRIMARY 20
$ db2 update db cfg using LOGSECOND 20

# 空间数据库要求APPLHEAPSZ最少2048
$ db2 update db cfg for geodb using APPLHEAPSZ 2048

$ db2 force application all
$ db2stop
$ db2start

2.7. 开启空间数据库¶

  • 开启空间化

  • 创建sde用户

  • 赋予sde权限

$ db2se enable_db geodb

# ArcGIS要求用户名必须是sde
$ groupadd -g 911 sdegroup
$ useradd -g sdegroup -u 811 -d /home/sde -m -s /bin/bash sde
# passwd sde

$ su - bgis
$ db2 connect to geodb
$ db2 grant dbadm on database to user sde
$ db2 update dbm cfg using sysmon_group sdegroup
