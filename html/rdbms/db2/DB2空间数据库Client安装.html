1. DB2空间数据库Client安装¶

1.1. Client for Linux¶

  • 解压安装

  • 配置环境变量

  • 创建用户实例

$ tar -zxvf v9.7fp11_linuxx64_rtcl.tar.gz
$ ./rtcl/./db2_install

$ vi /etc/profile
export DB2_HOME=/opt/ibm/db2/V9.7
export PATH=$DB2_HOME/bin:$PATH
$ source /etc/profile

# 客户端未创建实例，无法启用命令行
$ cd /opt/ibm/db2/V9.7/instance
$ ./db2
SQL10007N Message "-1390" could not be retrieved.  Reason code: "3".

$ ./db2idrop -f arcgis
$ ./db2icrt -u arcgis arcgis

Note

Linux上安装的Runtime Client需要为现有非root用户创建实例，否则无法使用命令行

1.2. ArcGIS Server with Client¶

  • 直连DB2配置

  • 重启ArcGIS服务

$ vi /home/arcgis/server/usr/init_user_param.sh
# For Direct Connect with DB2
export DB2_CLIENT_HOME=/opt/ibm/db2/V9.7
export DB2INSTANCE=arcgis
export PATH=$DB2_CLIENT_HOME/bin:$PATH
export LD_LIBRARY_PATH=$DB2_CLIENT_HOME/lib64:$LD_LIBRARY_PATH

$ /home/arcgis/server/stopserver.sh
$ /home/arcgis/server/startserver.sh

Note

需要在ArcGIS的初始化脚本中配置DB2客户端的直连信息， arcgis 即为客户端创建的用户实例

1.3. ArcGIS Desktop with Client¶

  • Windows安装DB2客户端

  • 配置编目node

  • ArcGIS连接数据库

  • 客户端启用空间数据库

  • ArcGIS catalog注册数据库

# 查看编目
$ db2 list node directory
$ db2 list db directory
# 删除编目
$ db2 uncatalog node xxx
$ db2 uncatalog db xxx
# 创建编目
$ db2 catalog tcpip node geonode remote 192.168.30.106 server 60000
$ db2 catalog db geodb at node geonode
$ db2 connect to geodb user sde using sde
